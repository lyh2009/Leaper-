### 开发前的配置要求
1. CMake 3.12(或更高版本)
2. Git 2.1(或更高版本)
###### 推荐编译器
1. Clang 18.1.8(或更高版本)
2. GCC 13.2(或更高版本)
3. Visual Studio 17 2022(或更高版本)
4. 注意: 支持C++17的通常都能编译并不是必须要使用推荐的编译器

### 编译步骤
###### Windows编译步骤
```
mkdir build
cd build
cmake ../
```
